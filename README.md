# Exploring-Visual-and-Interactive-Elements

This code creates a complete, multi-media fan webpage for the fictional band Harmony Heights. It features integrated images, a playable video performance, structural tour schedules, social media links, and a functional user feedback form.

I built this project as a hands-on assignment during my learning journey with the Cisco Networking Academy. Through building this layout, I mastered HTML document structure and semantic metadata tagging. I learned to manipulate essential web elements, handle asset fallbacks, and distinguish between structural attributes (type, id, name) for browser behavior, UI element linking, and form processing.
